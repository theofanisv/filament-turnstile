# Changelog

All notable changes to `filament-turnstile` will be documented in this file.

## v1.0.0 - 2025-11-15

### Added
- Initial release
- Cloudflare Turnstile integration for Filament v4
- Custom form component with fluent API
- Server-side validation with detailed error messages
- Automatic widget reset on validation errors
- Support for theme, size, and language customization
- Test mode for development
- Comprehensive documentation and examples
- Full test suite
- Multi-language support
